我的公众号历程

本以为最开始留下的自媒体坑位我不会有兴趣去填起来了，没想到我依然再次走上了自媒体的道路，那么就先来讲讲我的故事吧，虽然还没有开始码字，但是预感可能会比较长，感兴趣的朋友可以往下听，不感兴趣的就可以绕道走了，别无他求，不喜勿喷🙏。

整个故事分为五个模块讲：

  • 做公众号的缘由（记录个人心路历程，没什么干货，懒得看就跳过吧）

  • 第一个公众号是如何做大到毁灭的？

  • 短暂的第二个公众号

  • 为什么要做技术公众号？

  • 和朋友共同运营公众号

  • 未来？？？（这是第六个模板，给未来的可能预留的）

# 做公众号的缘由

2018年7月1日，是我毕业的时候，在毕业前就签约了中国水利水电第十一工程局，大国企，对于还没有步入社会，不知道自己想要什么的我而言，似乎感觉很棒，对于家中长辈而言，似乎也挺满意，毕竟传统观念上，进了国企就好像有了铁饭碗一样，一辈子不用发愁了一样，当时尚且天真的我也是这么想的。🤷‍♂️

但是好景不长，也许我天生对施工反感，也许是性格使然，我没办法和工人融入到一起（说到底就是该死的学生气息），在工地摸爬滚打两年都没能彻底洗脱我身上的学生气，这一点我也不知道如何评判，可能是生性固执，也可能是始终不愿意接受所谓的“现实”？总之，我就是无法融入到工地的生活中，很努力了，但是内心对那种状态的排斥，使得我越努力越痛苦。

直到，一次机缘巧合下认识了一个做公众号的大佬磊哥。磊哥人真的很好，通过他我接触到一片全新的世界，当时小白一个的我有很多很多的问题，他有时间的时候都比较认真地回答了我，并且得知我要开始做公众号之后，把我拉进了一个号主联盟的群（当时大概300多人，很多几十万粉的大佬在里面）。他和我说在群里多看看别人聊天，可以学到很多东西，公众号水很深，坑也很多，多了解了解，学一下怎么做，一边做一边完善。

就是这样一个群，改变了我的个认知和接下来做的每一个决定，可以说这是我人生的一个转折点。👀

# 第一个公众号

在接触到一批优秀的公众号号主有一段时间之后，我被这些多才多艺的号主折服了，感觉里面有很多人真的很强！！尤其是，当我自己也开始写公众号之后，对这些大佬更是致以崇高的敬意。🙏

# 想法萌生

**时间：**2019年7月的某个清晨🕖

**坐标：**湘西土家族苗族自治州 - 泸溪县 - 大山里的辛女溪旁

情景复现： 由于住的是临时搭建的彩钢房，所以平日里洗漱、洗碗等都是在门外修建的临时水池，湘西的雨水很多，尤其在7月的时候，几乎每天都是暴雨，连绵不休。就是这样一个普通的早上，我穿着雨衣，站在门外，刷牙杯中混着抽上来的地下水和雨水刷着牙。

平静的内心突然有一个声音在问我自己，这样的生活你喜欢吗？吃苦耐劳是美德，但是有必要这样吗？有什么意义吗？作为一个远走他乡的游子，放弃了城市的生活，放弃了我喜欢追求的一切，跟随项目部一头钻进了深山老林中，赚的钱也就是很普通的5000块，真的就要只为了一份看似稳定的工作，把一切都舍弃吗？.....

内心有无数疑问闪过，心里的不甘瞬间无数倍放大，在暴雨中刷牙的我突然抬起头，闭上了眼睛，任由雨水冲刷在脸上。我，不甘心！！😖

之所以会这么不甘心，很大的原因在于我接触到了一批优秀的公众号号主，见识到了他们的个人能力，见识到了他们的拼搏，见识到了原来赚钱可以那样轻松，这一切对当时懵懂的我冲击是那么巨大，可能内心本来就有不屈服的火苗，迎风变涨，最后把我吞噬。

# 开始实施

`很多人想法都很多，但是没有付诸行动，最终都变成了空想。`我也是个普通人，有过太多想法，也有太多想法只是一个空想而已，深知不行动就不会有改变，所以从那个时候开始，因为动力足够大，我开始不断提醒自己，`要当一个行动派，不能只是心里想想、嘴上说说。`

于是，就在那天晚上下班后，我就开始行动了——

  • 创建公众号
    • **怎么创建？**遇到什么就百度，我多年来的习惯都是如此，很少问别人问题，都是自行百度解决，所以做公众号也一样，先从百度如何创建公众号开始）
    • **类型选择？**创建过程里只遇到一个选择：公众号类型。百度之后我发现作为一个个体，只能选择订阅号，好吧，那就创建一个订阅号！！
    • **内容类型定位？**做公众号一定要有一个明确的定位！！那就是你要明确这个公众号主要是用来干嘛的：比如是分享情感文章，引起大家共鸣？分享有用咨询信息，能给一些人提供便利？或者分享一些优质的资源，让关注你的人能够节省找资源用的时间？抛出几个问题给大家，我的定位马上就会揭晓。
    • **公众号起名？**对于起名字这个问题，当时让我很是纠结，因为我很想很想做一个写优质文章的公众号，但是我清楚认识到自己并没有那样优秀的文笔，也没办法持续输出，所以选择了一个感觉比较容易发展的方向：分享资源！！这样只要有资源可以分享，就不愁没办法写出来文章了~名字嘛，就很直接了当起了一个：资源乐享吧。（中二少年🤣）
  • 文章排版
    • 作为一个处女座的人，经常对一些细小的东西各种不满意，阻挡我写文章的第一个拦路虎就是排版问题。当时不太会写东西，写一篇300字的文章都得用好久，但是真正消耗了大部分时间的其实不是写文章这件事本身，而是不那么重要的文章排版问题！！打个比方：如果一篇文章我花费了四个小时，那么其中只有半小时是在写文章，进行内容创作，剩余的三个半小时都耗费在了文章排版上！！比如挑选引导阅读的样式、挑选文末点击在看的样式、文章过度的样式、图片边框的样式、阴影效果的样式、文字的大小、字间距、行间距、行两端缩进、文章封面的配图......太多版式在干扰着我，让我不能专注于内容生产。所以我花费了一天的时间，制定了一个自己比较满意的模板，然后每次就使用那个模板进行二次创作，这样就不用在版式上浪费太多的时间了。

# 走向毁灭

公众号走向毁灭其实很简单，有句话说：“病来如山倒！”，其实公众号距离封号也类型，封号前几乎不会有任何征兆，只要触发到了那么一个点，引来的就是 永封。🏴‍☠️

这里提醒大家，在进行公众号运营之前，一定要熟读微信公众号运营规范 (opens new window)，每个行业都有自己的规范，既然依托于微信公众平台，就一定要遵守人家定好的规矩，小心谨慎都可能阴沟里翻船，更不要说“明知故犯，走在'河边' ”。

这是我在公众号被封号之后才去正视的一份规范，当我阅读了这份规范之后发现所有的封号、账号功能限制、警告等都是有迹可循的，公众号平台的流量很大，所以必然要采取严格的管控措施，不然可能走向毁灭的就不是某个人的公众号，而是这个平台了。在做号前一定要认清这一个本质，平台的管控，是因为法律的约束，平台不想走向毁灭，就一定会立有很多规矩，不遵守规矩就是想要让平台陪葬，你觉得平台会陪你嘛？？

# 短暂的第二个公众号

第二个公众号非常短暂，只做了一个月的时间，可以说昙花一现。当时是我第一个公众号，也就是我的本命大号被封禁的时候，短暂的懵圈之后，我觉得万幸的是我还有一个引流过两天的小号存在，不至于大半年的努力全部付诸东流。所以本着不服输的态度开始了第二个公众号了的写作。

# 个人技术公众号

做一个个人技术公众号是我在做资源号的时候一直在想的一件事，因为分享资源，其实就是充当网络搬运工，进行一些现有资源的尝试、判断、汇总，当把整套流程都很熟练之后，对个人能力的提升就约等于无了。我是一个很有危机感的人，当我很久没有感到自己在进步的时候就会感到恐慌。😱

所以我觉得应该做一些能够对自己本身有帮助的公众号，就比如技术总结类型的号，就算没有那么多人看，但是是对自己本身的一种升华。有时候只要感觉到自己在进步，就会很安心、很踏实、很满足。

曾子曰：“吾日三省吾身：为人谋而不忠乎？与朋友交而不信乎？传不习乎？” 写技术博客、技术总结公众号，不仅是做自媒体，还是一种对知识的温习。尤其对于学习计算机技术而言，如果只是看了一遍，ok，你只是知道了有这么个技术，真让你自己总结性说一下这个技术功能是什么？它是如何实现的？具体有哪些操作指令？有什么注解？光是这些浅层次的东西估计都没办法回答出来个七七八八，更别说什么“深入了解底层原理，熟悉各种算法，熟练使用常用的设计模式等等”更加有深度、有难度的问题了。

子曰：“温故而知新，可以为师矣”。当我在写技术博客的时候，通常相当于把刚刚看到的技术点、知识点看第二次，最简单直白的是看第二次，并且对照着敲下来，还有时候会进行一些简单的思路梳理，画一下流程图、思维导图、或者用一个框图将所有的模块进行关联和区分，这是一个很缓慢的过程，正是这样一个缓慢的“复习”过程，就能够有很多新的体会、新的发现，对一个知识点的了解更进一步。

真的是经历越多，越是感慨古代先贤的智慧，很多话语都言简意赅地指出了世界的根本规律。

# 和朋友共同运营公众号

在我做公众号的过程里认识了很多优秀的自媒体号主，其中有一个同是山西的老乡，最为聊得来。而且认识的时间也非常久了，从我第一个公众号阅读不到100的时候就认识了，期间经常会有沟通。

他曾经说过一句让我很感动的话，“怎么说没就没了呢，你的公众号我看着它发展起来的，从最开始不到100阅读，到几百阅读，1000阅读，然后阅读超过我，再然后比我阅读高了两倍，这突然就没了，感觉我都接受不了，心里好难受啊！”。这是我的账号被封之后，在号主交流群里有人发现我的账号被封号，截了个图说，“又一个老兄凉凉了，哎~”。他看到这个消息之后就立马来问我怎么回事，多久解封，当得知是永久封禁的时候说的话，当时我整个人是懵逼的，也不想说什么，也不想难过，就什么都不去思考，很淡定地聊了聊，表示我没事。

说实话对于做软件资源分享，我的心里并不看好，因为这件事任何人只要不懒惰都可以做，而且收集、测试、写对应软件的推文是一件很耗费时间的事情，所以当我不做公众号很久，他和我说他之前的号卖掉了，想重新开始，和我一起弄一个号，这样大家都轻松的时候，我并没有很想做
