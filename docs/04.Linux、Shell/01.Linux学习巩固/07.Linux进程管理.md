## Linux中的进程类型：

Linux操作系统包括3种不同类型的进程：

1、`交互进程`：是一个Shell启动的进程，可以在前台或后台运行；
2、`批处理进程`：与终端没有联系，是一个进程序列；
3、`监控进程（守护进程）`：在系统启动时启动，并保持在后台运行。
