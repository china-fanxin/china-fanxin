# 博客写作

通常来讲，大家最常用的写作软件应该是Word、Text,可能有些人还没听过MarkDown,我这里推荐大家写博客必备语法`MarkDown`。

## MarkDown是什么？

Markdown是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。

## MarkDown的优势

- 纯文本格式写作
- 简洁、高效、易读、易写
- 可与HTML混编
- 可在MarkDown内写Vue语法
